DENON app icon resized assets

- AppIcon-1024x1024.png: App Store / master icon
- iOS_AppIcon.appiconset: can be dragged into Xcode Assets.xcassets for iPhone/iPad
- macOS_AppIcon.iconset: PNG iconset for macOS app icons

Note: macOS .icns generation requires Apple's iconutil on macOS:
  iconutil -c icns macOS_AppIcon.iconset
